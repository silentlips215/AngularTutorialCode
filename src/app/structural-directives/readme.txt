1- Structural directives are responsible for the Structural and layout of the DOM Element.
2- It is used to hide or display the things on the DOM.
3- There are different types of Structural Directives in Angular.
    (i)- ngif
    (ii)- ngif-else
    (iii)- ngSwitch
    (iv)- ngFor



What is ng-template?
ng-template is an Angular directive used to define reusable HTML content without rendering it directly. We use it to conditionally display or dynamically insert content in components.
