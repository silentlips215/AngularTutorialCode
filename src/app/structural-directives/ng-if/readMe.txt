1- ngIf directive is used to display or hide the DOM Element based on the expression value assigned to it.
2- The expression value may be either true or false.
3- The ngIf Directive is used to add ore remove HTML Elements according to the expression.
4- The expression must return a Boolearn value.
5- If the expression is false then the element is removed, otherwise element is inserted.
6- The ngIf directive does not hide the DOM element. It removes the entire element along with its subtree from the DOM.
